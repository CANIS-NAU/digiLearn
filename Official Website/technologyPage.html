The Technologies

The following is a summary of the major technologies we will use to make the Digital Backpack a reality. This page will continue to be updated as we work on our project.

  • Language: For languages we are planning on using Java and Python to develop the software as those are the languages the group is skilled in. We are also planning on using Kotlin for the mobile application that can easily be used for creating content for android and ios.



  • UI Development: Adobe Experience Design, or Adobe Xd, is a powerful tool for the development of functional and attractive graphical user interfaces. We chose Xd for it's ability to produce high-quality interfaces and its support for cross-platform development.



  • Cross-Platform Development: Flutter is a platform that enables synchronous, cross-platform development for Android, iOS, and web applications. We chose Flutter because it reaches all of the platforms that we need while also boasting better performance and easier maintenance when compared to the alternatives.



  • Networking:We have chosen Digital Ocean as the service on which to host our proxy server. Digital Ocean is both affordable and meets all of the needs of our project at this stage. The proxy server will be written using the Django Python framework. Django is a popular and well-supported option, and its extensive documentation and built-in features will streamline the development of the proxy server.
